#SNS+社区问答类网站

Springboot+Mybatis+Thymeleaf开发，数据库使用Mysql+redis，异步框架处理消息推送，同时使用了Pyspider爬虫进行网站内容数据填充。

***

**功能**

[1. 用户注册登录管理](#用户注册登录管理)

[2. 问题发布 敏感词过滤 问题广场](#问题发布 敏感词过滤 问题广场)

[3. 评论中心 站内信](#评论中心 站内信)

[4. Redis实现赞踩功能](#Redis实现赞踩功能)
